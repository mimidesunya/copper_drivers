CTI ドライバ Python版
バージョン @version@

Python(http://www.python.org/)を使ってCopper PDF 2.1以降にアクセスするためのプログラムです。
使用方法は付属のAPIドキュメント、サンプルプログラムまたは以下オンラインマニュアルを参照してください。
http://dl.cssj.jp/docs/copper/3.0/html/3425_ctip2_python.html

■ 動作環境
Python 2.4.3以降

■ 付属物

code           -- ドライバ本体(ソース)
apidoc         -- APIドキュメント(pydoc)
test/python2.4 -- サンプル・プログラム(Python2.6以前)
test/python2.7 -- サンプル・プログラム(Python2.7以降)
